Questions

Why is textContent safer than innerHTML?

textContent is safer because it only adds plain text to the page. It won't run HTML or JavaScript code, which helps prevent security problems.

How are HTML elements connected to JavaScript variables using DOM selection?

JavaScript uses DOM selectors like document.querySelector() or document.getElementById() to find an HTML element and save it into a variable so it can be changed later.

What would happen to the text box with id="text" if I added this line to the code above it: text.textContent = "Hello!"?

The element with the id of text would display the word "Hello!" instead of whatever text was there before.
